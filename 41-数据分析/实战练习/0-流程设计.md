# 流程

## 基本流程

- 明确问题
  - 描述性分析
  - 预测性分析
- 抓取数据
  - 字段设计
  - 数据提取
- 数据清洗
  - 异常值
  - 空白值
  - 无效值
  - 重复值
- 数据整理
  - 统一格式
- 建模分析
  - 变化
  - 分布
  - 对比
  - 预测
- 洞察结论
- 撰写报告
  - 报告背景 业务状况
  - 报告目的 解决什么问题
  - 数据基本情况 数据可信度
  - 可视化图表 数据的可理解程度
  - 策略选择 提出解决方案

## 建表

```sql
CREATE TABLE `gold`.`record` (
  `id` INT NOT NULL AUTO_INCREMENT,
  `metal_type` VARCHAR(45) NULL,
  `is_buy` INT NULL,
  `price` FLOAT NULL,
  `number` FLOAT NULL,
  `money_type` VARCHAR(45) NULL,
  `total_price` FLOAT NULL,
  PRIMARY KEY (`id`),
  UNIQUE INDEX `idnew_table_UNIQUE` (`id` ASC) VISIBLE);
```
